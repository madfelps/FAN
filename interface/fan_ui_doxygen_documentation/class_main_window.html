void MainWindow::sendJsonToUDP ( const QJsonObject &  qJsonObject)
private

This function sends a datagram formatted in JSON to socket.

If you need to send a JSON message, for example {"ID":0x01,"Speed": 100}, do this:

  1. create a object: QJsonObject Data_packet;
  2. fill in the message: Data_packet["ID"] = "0x01"; Data_packet["Speed"] = "100";
  3. send JSON message using this function: sendJsonToUDP(Data_packet);
