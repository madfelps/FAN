Detailed Description

The speedometer widget class.

If you are interested in introducing another speedometer in the interface, just go to mainwindow.ui and introduce a widget block (search for items available by Widget) and promote it to QMeter (right-click and go to Promove To >, there must be one option called QMeter, which is the created widget).
